# Gunacode

![Gunacode](images/gunacode_icon.png)

A theme for VS Code inspired by Guna ([https://github.com/poucotm/Guna](https://github.com/poucotm/Guna))

This extension includes two variants:
- Gunacode Dark
- Gunacode Light

Gunacode Dark:
![Gunacode Dark — Editor](images/gunacode_dark.png)

Gunacode Light:
![Gunacode Light — Editor](images/gunacode_light.png)

## Installation
Install from the VS Code Extensions Marketplace.
